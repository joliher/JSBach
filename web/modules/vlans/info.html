📖 Módulo VLANs - Manual de Usuario

Este módulo permite segmentar la red física en múltiples redes lógicas independientes para mejorar la seguridad y eficiencia.

🚀 Control del Servicio

Gestione el ciclo de vida de las interfaces virtuales:

  • START: Levanta todas las interfaces VLAN configuradas.
  • STOP: Desactiva las interfaces VLAN (cuidado: puede afectar a la conectividad).
  • RESTART: Reinicia el servicio para aplicar nuevos cambios de IDs o IPs.

⚙️ Configuración de VLANs

Cada VLAN requiere:

  • ID VLAN: Un número entre 1 y 4094.
  • Nombre: Una etiqueta descriptiva (ej: VoIP, Guest, CCTV).
  • IP Interfaz: La dirección IP del router en esa VLAN (ej: 192.168.20.1/24).
  • IP Red: El identificador de red (ej: 192.168.20.0/24).
⚠️ RESTRICCIÓN: Las VLANs 1 y 2 están reservadas por el sistema para gestión y servicios críticos. No intente eliminarlas.
