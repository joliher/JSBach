📖 Módulo Firewall - Manual de Usuario

Este módulo gestiona la seguridad entre redes (VLANs) y el acceso al propio router mediante un sistema de **Whitelist Dinámica**.

🔐 Conceptos de Seguridad

  • Whitelist: Por defecto, el firewall deniega todo tráfico que no esté explícitamente permitido en esta lista.
  • Aislamiento (Isolated): Si está activo, los dispositivos de esta VLAN no podrán comunicarse con ninguna otra VLAN de la red.
  • Restricción (Restricted): Si está activo, los dispositivos de esta VLAN tendrán denegado el acceso administrativo al router.
  • Allowed IPs: Direcciones IP específicas que están exentas del bloqueo y pueden acceder al router incluso si la restricción está activa (útil para IPs de gestión).

👁️ Visualización

En la sección VER VLANS puede comprobar de un vistazo el nivel de blindaje actual de su infraestructura.

⚠️ DEPENDENCIA CRÍTICA: El Firewall depende de que las interfaces VLAN estén correctamente configuradas e iniciadas. Sin el módulo VLANs activo, las políticas de seguridad no podrán aplicarse sobre el tráfico segmentado.
