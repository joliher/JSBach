📖 Módulo WAN - Manual de Usuario

Este módulo permite gestionar la conectividad a Internet de la interfaz externa del router.

🚀 Acciones Rápidas

A través de la barra lateral puede controlar el ciclo de vida del servicio WAN:

  • START: Inicia la interfaz y solicita IP (DHCP o estática).
  • STOP: Desactiva la interfaz y limpia las rutas.
  • RESTART: Reinicia el servicio para aplicar cambios o refrescar la concesión DHCP.
  • STATUS: Muestra el estado actual de la interfaz y la tabla de rutas.

⚙️ Configuración

En la sección CONFIG puede definir cómo se comporta la interfaz:

  • DHCP: Modo recomendado para la mayoría de conexiones domésticas y de oficina.
  • Manual/Estático: Permite definir IP, máscara, gateway y DNS cuando el proveedor así lo requiere.
⚠️ PRECAUCIÓN: Una configuración incorrecta de la interfaz WAN puede dejar al router sin acceso a Internet y afectar a todos los clientes de la red local.
