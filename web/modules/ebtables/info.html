📖 Módulo Ebtables - Manual de Usuario

Este módulo proporciona seguridad granular a nivel de Capa 2 (Bridge) utilizando el framework ebtables del kernel Linux.

🚀 Control del Aislamiento

  • START: Inicia el motor ebtables y aplica todas las reglas de aislamiento PVLAN y filtrado MAC.
  • STOP: Desactiva ebtables y limpia todas las cadenas de filtrado de puente.
  • RESTART: Reinicia el servicio para refrescar las tablas de acceso y reglas dinámicas.
  • STATUS: Muestra las reglas activas en la tabla 'filter' de ebtables.

🛡️ Private VLAN (PVLAN)

El aislamiento PVLAN impide que los dispositivos dentro de una misma VLAN se comuniquen entre sí o con otras VLANs a nivel de bridge, permitiendo únicamente el tráfico hacia la puerta de enlace (Gateway) para acceso externo (WAN).

⚠️ NOTA DE RENDIMIENTO: El filtrado por ebtables consume recursos del procesador en el puente de red. Utilice el aislamiento PVLAN solo en las redes que realmente requieran segregación estricta de clientes (ej: redes de invitados o servidores DMZ).
