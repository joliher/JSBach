📖 Módulo Tagging - Manual de Usuario

Este módulo permite gestionar el etiquetado de tráfico (VLAN Tagging) en las interfaces físicas del router utilizando el estándar IEEE 802.1Q.

🚀 Operaciones Principales

  • START: Aplica todas las configuraciones de etiquetado a las interfaces físicas.
  • STOP: Limpia todas las reglas de VLAN en las interfaces integradas.
  • RESTART: Reinicia el servicio para aplicar cambios en los modos de acceso o troncal.

⚙️ Modos de Configuración

Cada interfaz física puede configurarse de dos maneras:

  • UNTAG (Modo Acceso): La interfaz procesa el tráfico de una sola VLAN de forma transparente para los dispositivos conectados.
  • TAG (Modo Troncal/Trunk): La interfaz transporta paquetes con etiqueta de múltiples VLANs, permitiendo la conexión a otros switches o routers gestionables.
⚠️ RESTRICCIÓN: Una interfaz física no puede ser UNTAG y TAG al mismo tiempo para el mismo rango de IDs. Asegúrese de que las VLANs especificadas hayan sido creadas previamente en el módulo VLANs.
