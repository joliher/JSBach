📖 Módulo Expect - Manual de Usuario

Este módulo centraliza la orquestación remota de dispositivos de red mediante scripts automatizados.

🚀 Orquestación

La orquestación permite enviar comandos de configuración a múltiples switches de forma simultánea o secuencial. Puede configurar puertos, VLANs o parámetros globales del sistema.

🧩 Gestión de Switches

En la sección SWITCHES puede registrar sus dispositivos:

  • Protocolos: Soporte para Telnet y SSH.
  • Perfiles: Adaptadores específicos para Cisco IOS, TP-Link, etc.
  • Credenciales: Nombre de usuario y contraseña (o IP sin clave).

📶 Control MAC

Permite visualizar la tabla dinámica del switch y realizar Aislamientos. El sistema mantiene un registro local de las MACs bloqueadas para asegurar que la regla persista aunque el switch reinicie o limpie su tabla dinámica.

⚠️ PRECAUCIÓN: Asegúrese de tener conectividad IP con los switches antes de iniciar una tarea de orquestación masiva.
