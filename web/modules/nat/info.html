📖 Módulo NAT - Manual de Usuario

El módulo NAT (Network Address Translation) permite que los dispositivos de su red local compartan la conexión a Internet del router mediante la técnica de enmascaramiento (Masquerading).

🚀 Operaciones del Servicio

  • START: Activa el reenvío de paquetes (IPv4 Forwarding) y establece las reglas de Masquerading en la interfaz exterior.
  • STOP: Elimina las reglas de NAT y desactiva el reenvío de paquetes.
  • STATUS: Muestra las reglas actuales en la tabla 'nat' de iptables y verifica el estado de forwarding del kernel.

⚙️ Configuración

Para que el NAT funcione correctamente, debe indicar qué interfaz está conectada a la red externa (WAN). Generalmente es la interfaz configurada en el módulo WAN.

⚠️ REQUISITO: El servicio NAT depende del módulo WAN. Si el módulo WAN no está activo, el NAT no podrá proporcionar salida a Internet aunque el servicio esté iniciado.
